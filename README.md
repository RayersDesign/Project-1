# ⛳ Fairway & Co. — Landing Page

Landing page pour **Fairway & Co.**, une marque de vêtements de golf vintage et seconde main, style country club.

## Contenu

- **Hero** — Accroche visuelle avec CTA
- **À propos** — Histoire de la marque
- **Collection** — Grille produits avec filtres par catégorie
- **Pourquoi Vintage** — Arguments durabilité + style
- **Newsletter** — Inscription email
- **Footer** — Liens et réseaux sociaux

## Stack

HTML5 · CSS3 · Vanilla JS (aucune dépendance)

## Lancer localement

Ouvrir `index.html` directement dans un navigateur, ou avec un serveur local :

```bash
npx serve .
# ou
python -m http.server 8000
```

## Personnalisation

- Remplacer les placeholders de produits par de vraies photos
- Connecter le formulaire newsletter (Mailchimp, Brevo, etc.)
- Adapter le nom de marque, les couleurs dans les variables CSS (`:root` dans `style.css`)
